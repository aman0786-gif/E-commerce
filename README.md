# WearHive E-Commerce Website

A responsive e-commerce website built using HTML, CSS, and JavaScript.

## Features

- Responsive Navbar
- Product Categories
- Featured Products
- Latest Products
- Special Offer Section
- Customer Testimonials
- Footer with Social Links
- Mobile Menu Toggle

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome

## Folder Structure

```bash
project/
│
├── index.html
├── style.css
└── images/
```

## Run Project

1. Download the project.
2. Open `index.html` in your browser.
